# 富文本编辑器使用指南

## 📝 功能特性

### 1. 文本格式化
- **粗体** - 点击 B 按钮或选中文字后点击
- *斜体* - 点击 I 按钮
- <u>下划线</u> - 点击 U 按钮
- 标题 1 和标题 2 - 用于文章结构

### 2. 列表和引用
- 无序列表 - 项目符号列表
- 有序列表 - 数字编号列表
- 引用块 - 引用他人的话或重要内容

### 3. 链接和代码
- 插入链接 - 点击链接图标，输入 URL
- 代码块 - 用于展示代码片段

### 4. 图片上传（三种方式）

#### 方式 1：点击上传
1. 点击工具栏的图片图标 📷
2. 选择本地图片文件
3. 图片会自动上传并插入到编辑器

#### 方式 2：粘贴图片
1. 复制图片到剪贴板（截图、复制图片等）
2. 在编辑器中按 `Ctrl+V` (Windows) 或 `Cmd+V` (Mac)
3. 图片会自动上传并插入

#### 方式 3：拖拽图片
1. 从文件管理器拖拽图片文件
2. 放到编辑器区域
3. 图片会自动上传并插入

### 5. 撤销和重做
- 撤销 - 点击 ↶ 按钮
- 重做 - 点击 ↷ 按钮

## 🎨 使用场景

### 文章编辑
- 新建文章时使用富文本编辑器编写内容
- 支持中英文双语内容编辑
- 可以插入图片、代码、链接等

### 项目详情
- 项目管理页面现在支持富文本详细内容
- 可以详细介绍项目的功能、技术栈、特色等
- 支持图文混排

### 关于页面
- 个人简介支持富文本格式
- 可以添加图片、链接等丰富内容

## 📦 图片上传优化

### 延迟上传模式（默认）
- 选择图片后，先显示本地预览
- 点击"保存"按钮时才真正上传到服务器
- 优点：速度快，用户体验好
- 适用场景：项目管理、关于页面等

### 立即上传模式
- 选择图片后立即上传到服务器
- 适用于需要立即获取图片 URL 的场景
- 适用场景：文章封面图等

## 🔧 配置说明

### 图片存储配置
1. 打开 Supabase Dashboard
2. 进入 SQL Editor
3. 运行 `supabase-storage-setup.sql` 脚本
4. 刷新页面，在 Storage 中应该能看到 images 桶

### 项目表更新
如果项目管理页面需要详细内容字段：
1. 打开 Supabase Dashboard
2. 进入 SQL Editor
3. 运行 `supabase-projects-update.sql` 脚本

## 🐛 常见问题

### Q: 图片上传失败怎么办？
A: 
1. 检查是否运行了 `supabase-storage-setup.sql`
2. 检查 Supabase Storage 是否配置正确
3. 可以使用图床服务（如 imgbb.com）上传图片，然后使用 URL 方式

### Q: 更换图片按钮没反应？
A: 已修复！现在点击"更换图片"按钮会正确触发文件选择。

### Q: 上传速度慢怎么办？
A: 
1. 使用延迟上传模式（默认）
2. 压缩图片后再上传
3. 使用图床服务

### Q: 如何使用图床？
A: 
1. 推荐图床：imgbb.com、sm.ms、imgur.com
2. 上传图片到图床获取 URL
3. 在编辑器中点击"使用图片 URL"
4. 粘贴 URL 即可

## 🎯 最佳实践

1. **文章结构**
   - 使用标题 1 作为主标题
   - 使用标题 2 作为小节标题
   - 使用列表来组织要点

2. **图片使用**
   - 在文章开头使用封面图
   - 在内容中适当插入配图
   - 图片要有意义，不要过多

3. **代码展示**
   - 使用代码块展示代码
   - 添加必要的注释
   - 保持代码简洁

4. **链接使用**
   - 为重要的参考资料添加链接
   - 链接文字要有描述性
   - 避免使用"点击这里"等模糊文字
